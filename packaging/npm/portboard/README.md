# PortBoard for npm

This package installs the native PortBoard executable for macOS, Linux, or
Windows. It does not install Python and does not modify an existing Python
environment.

```bash
npx @leonpure/portboard
```

For a persistent installation:

```bash
npm install --global @leonpure/portboard
portboard
```

The source code, documentation, and alternative installation methods are at
<https://github.com/LeonPure/portboard>.
